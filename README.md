# Project Title
**N8N Automation**

## Deskripsi
N8N Automation adalah platform workflow automation yang bisa mengotomasi atau menghubungkan berbagai aplikasi, layanan, dan API.

## Ket Project
- Catatan pengeluaran.json untuk melihat catatan pengeluaran bisa memasukan berdasarkan input atau struk
- RAG Supabase.json untuk menjawab pertanyaan berdasarkan data yang ada seperti pdf 
- Text,Image,Voice.json untuk Text menjawab pertanyaan dari data seperti pdf atau lainnya, Image bisa menganalisis image dari caption yang diberikan, Voice bisa memberikan jawaban menggunakan Voice, jika pertanyaan tersebut pakai voice juga
- _FAQ.json untuk FAQ terkait data yang ada pada Sheet
- lyrics music.json untuk melihat lirik lagu berdasarkan inputan judul dan artis

Dibuat Menggunakan:  
> N8N Automation.
